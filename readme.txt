A really simple plugin for displaying WordPress memory utilization, server's uptime and Cpu last minute load on the new WordPress admin toolbar.

== Description ==

WP Really Simple Health is a WordPress plugin that allows you to view WordPress memory utilization, server's uptime and Cpu last minute load on the new WordPress admin toolbar. By default plugin shows all these three items, but you can only select items that interest you through the options page.
It uses classes to avoid naming collisions.

Developed by <a href="http://www.tautologicalcode.net">Tiberius14</a>

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

   1 - Unzip the archive and put the wp-really-simple-health folder into your plugins folder (/wp-content/plugins/).
   2 -  Activate the plugin from the Plugins menu.
